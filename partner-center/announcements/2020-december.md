## <a name="an-update-of-the-us-microsoft-365-business-voice-with-calling-plan-offer-is-coming-soon"></a><a name="2"></a>Uma atualização do US Microsoft 365 Business Voice com oferta de Plano de Chamada estará disponível em breve

### <a name="categories"></a>Categorias

- Data: 2020-12-3
- Ofertas/mercados

### <a name="summary"></a>Resumo 

Em 1º de janeiro de 2021, a Microsoft começará a fazer a transição de parceiros e clientes dos EUA para uma nova oferta do Microsoft 365 Business Voice com Plano de Chamada. Nenhuma ação do parceiro é necessária.

### <a name="impacted-audience"></a>Público-alvo afetado 

Todos os parceiros que fazem transações pelo programa CSP (Provedor de Soluções na Nuvem)

### <a name="details"></a>Detalhes

A Microsoft substituirá a oferta existente do Business Voice com Plano de Chamada nos EUA por uma nova oferta para dar suporte a requisitos internos para produtos de telecomunicações. A nova oferta terá o mesmo conjunto de recursos e preços. As alterações da oferta são internas à Microsoft e não devem afetar a forma como os parceiros de CSP comercializam, vendem ou dão suporte à oferta do Business Voice com Plano de Chamada. A alteração será aplicada somente a esta oferta.

Para novas vendas, você usará a nova oferta a partir de 1º de janeiro de 2021. A oferta antiga será substituída pela nova oferta na lista de preços.

Os clientes existentes serão automaticamente transferidos para a nova oferta após sua data de renovação. Nenhuma ação é necessária dos parceiros e dos clientes de renovação.

## <a name="now-live-partner-center-api-updates-and-user-interface-enhancements-for-the-education-customer-validation-process"></a><a name="1"></a>Ao vivo agora: atualizações de API do Partner Center e aprimoramentos da interface do usuário para obter o processo de validação do cliente de Educação

### <a name="categories"></a>Categorias

- Data: 2020-12-3
- Funcionalidades

### <a name="impacted-audience"></a>Público-alvo afetado 

Os parceiros que vendem ofertas Acadêmicas por meio do programa Provedor de Soluções na Nuvem

### <a name="summary"></a>Resumo 

O trabalho da Microsoft é baseado em confiança. Temos o compromisso de fornecer um método em conformidade e seguro de validação de clientes para realizar transações de ofertas Acadêmicas no programa Provedor de Soluções na Nuvem. Como parte desse compromisso, introduziremos a API do Partner Center e os aprimoramentos na interface do usuário, o que adicionará clareza e visibilidade ao processo de habilitação do cliente, bem como a capacidade de inserir dados mais precisos para aumentar o sucesso na habilitação do cliente. 

**Aprimoramentos do Partner Center** 

- Novas APIs de Qualificações GET e POST para dar suporte à entrada de dados precisos e aprimorar o processo de validação do cliente de Educação da Microsoft. 

- Aprimoramentos na interface do usuário para dar suporte à entrada de dados precisos e aprimorar o processo de validação do cliente de Educação realizado pela Microsoft. 

**Substituição das APIs de qualificação GET e PUT existentes** 

As APIs de qualificação GET e PUT existentes serão desativadas **antes do fim de fevereiro de 2021**. Nesse momento, você precisará fazer a transição para as novas APIs GET e POST do Partner Center para poder comprar ofertas para Educação.  

Testes. 

Para entender melhor as APIs e a entrada de dados necessárias para obter êxito na validação do cliente, os parceiros podem testar esses aprimoramentos. Os parceiros que desejam participar do teste deverão baixar o [guia de testes do cliente de Educação do Partner Center](https://partner.microsoft.com/resources/detail/partner-center-edu-testing-guide-pdf) para obter instruções sobre como se preparar, como se inscrever e o que esperar durante a fase de testes.

**Clientes de biblioteca e museu** 

Além dos aprimoramentos acima, estamos empolgados em anunciar que habilitaremos ofertas com preços Acadêmicos para clientes de biblioteca e museu, expandindo os clientes de Educação com os quais você pode realizar transações de ofertas CSP. 

A Microsoft reserva a si o direito de examinar o status de qualquer cliente ou cliente proposto como um usuário de Educação qualificado. Confira os [Requisitos do Usuário de Educação Acadêmica Qualificada](https://www.microsoftvolumelicensing.com/DocumentSearch.aspx?Mode=3&DocumentTypeId=7) para obter detalhes completos.
