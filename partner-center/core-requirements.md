# <a name="incentives-co-op-claims-core-requirements"></a>Solicitações principais de reivindicações de cooperação de incentivos

**Aplica-se a**

- Partner Center

**Funções apropriadas**

- Administrador de incentivos
- Usuário de incentivos

Cada tipo de atividade para a qual você envia uma declaração de incentivos tem requisitos básicos. Você precisa demonstrar que atendeu a esses requisitos nos documentos comprovantes de execução. Algumas atividades também exigem tipos de comunicação e métricas específicos.

As tabelas a seguir descrevem os requisitos básicos para cada uma das atividades aceitas. 

## <a name="demand-generation"></a>Geração de demanda

### <a name="print-advertising"></a>Anúncio impresso

| Categoria | Comprovante de execução    |
| ------ | ----------- |
| Requisitos básicos    | Marca da empresa e informações de contato<br><br>Conteúdo da Microsoft |
| Requisitos de comunicações | Cópia da peça de comunicação final (com base no anúncio) mostrando requisitos básicos específicos do programa |
| Métricas     | Alcance estimado de cliente (ou parceiro) |
|Despesas qualificadas    | Colocação de mídia de terceiros<br><br>Material impresso<br><br>Serviços de marketing e criativos, incluindo tarifas de agência  |

### <a name="digital-advertising"></a>Anúncio digital

| Categoria | Comprovante de execução    |
| ------ | ----------- |
| Requisitos básicos | NA |
| Requisitos de comunicações | Captura de tela do anúncio digital<br><br>AND<br><br>URL da página de aterrissagem, mostrando os requisitos básicos específicos do programa |
| Métricas     | Resumo dos resultados (como o relatório navegável por cliques) |
|Despesas qualificadas    | Colocação de mídia de terceiros<br><br>Tarifa – agência: Serviços de marketing e criativos, incluindo taxas de agência |

### <a name="direct-mail-email-and-short-message-service-sms"></a>Mala direta, email e SMS

| Categoria | Comprovante de execução    |
| ------ | ----------- |
| Requisitos básicos    | NA |
| Requisitos de comunicações | 1) Cópia do artigo de comunicação final mostrando requisitos básicos específicos do programa<br><br>2) A comprovação de distribuição pode incluir envelope de remessa carimbado ou outra comprovação de correspondência (mala direta), email final mostrando o cabeçalho datado ou outra comprovação de envio (email) ou foto de mensagem de telefone (SMS) |
| Métricas     | Alcance estimado de cliente (ou parceiro) |
|Despesas qualificadas    | Aquisição de banco de dados<br><br>Material impresso<br><br>Postagem<br><br>Tarifa – agência: Serviços de marketing e criativos, incluindo taxas de agência |

### <a name="partner-website-and-search-engine"></a>Site e mecanismo de pesquisa do parceiro

| Categoria | Comprovante de execução    |
| ------ | ----------- |
| Requisitos básicos    | NA |
| Requisitos de comunicações | URL e/ou capturas de tela de páginas do site mostrando a URL<br><br>Declaração de trabalho ou contrato de serviço para projeto SEO (somente atividade SEO) |

|Despesas qualificadas    | Tarifa – agência: Serviços de marketing, incluindo taxas de agência<br><br>Programação/desenvolvimento: Tarifas de programação e desenvolvimento<br><br>Tarifa – SEO: Taxas de otimização e manutenção  |

### <a name="microsoft-syndicated-content"></a>Conteúdo sindicalizado da Microsoft

| Categoria | Comprovante de execução    |
| ------ | ----------- |

| Requisitos de comunicações | URL e/ou capturas de tela de páginas do site mostrando a URL |

### <a name="multi-touch-digital-campaign"></a>Campanha digital multitoque

| Categoria | Comprovante de execução    |
| ------ | ----------- |
| Requisitos básicos    |NA |
| Requisitos de comunicações | Todos os comprovantes de execução, conforme detalhado na página da atividade aplicável, são necessários<br><br>Anúncio digital<br><br>Email direto<br><br>Site de Parceiro e Otimização do Mecanismo de Pesquisa<br><br>Telemarketing  |
